## Key Takeaways

- Autoresearch's significance is the pattern, not the results: the scientific method compressed into a loop agents can execute indefinitely, applicable far beyond neural network training
- The loop works because it enforces discipline (one change at a time, hypothesis before experiment) and memory (git history as lab notebook) -- without these constraints, agents make chaotic multi-variable changes
- Two emerging loop flavors: closed-loop optimization with defined endpoints (AlphaEvolve) and open-ended research without finish lines (Autoresearch)
- PraxLab is an open-source harness with a 5-command SQLite-based CLI memory layer; 4 Claude Code instances ran 550+ experiments over 48 hours with zero intervention
- Key PraxLab results: RL hitting 93% on MATH level 4-5, self-distillation matching RL at half cost, tool routing improving from 0.76 to 0.94
- As experimental costs approach zero, the human bottleneck shifts to strategic decisions: which task, which metric, which constraints to set before the loop starts

## One-Line Summary

The autoresearch pattern -- discipline plus structured memory in a tight agent loop -- transforms coding agents into autonomous researchers, as demonstrated by PraxLab's 550+ unattended experiments achieving state-of-the-art results.
