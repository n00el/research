# Everything Claude Code Can Do That You Haven't Tried Yet

**Author:** Sharbel (@sharbel)
**Date:** March 10, 2026
**Source:** https://x.com/sharbel/status/2031372411283575012

## Key Takeaways

- Agent Skills (SKILL.md folders) let you permanently teach Claude Code your workflows, conventions, and deployment processes — now an open standard at agentskills.io
- SessionStart hooks automatically inject fresh context (branch, decisions, rules) after /compact or session restart, preventing context loss in long sessions
- Claude Code can audit unmanaged AWS resources and generate + import CloudFormation templates in a single session, handling dependency ordering automatically
- GSD (Get Shit Done) is an external tool that fixes context rot — the quality degradation after 3-4 tasks when the context window fills with stale tool outputs (25.9K GitHub stars)
- Google's new official Workspace CLI gives agents native access to Gmail, Calendar, Drive, and Docs without violating ToS or risking account bans
- The common pattern: treating Claude Code as a programmable agent runtime rather than a coding autocomplete is the key differentiator

## One-Line Summary

Five underused Claude Code capabilities — Agent Skills, SessionStart hooks, CloudFormation import, GSD context management, and Google Workspace CLI — that transform it from a code autocomplete into a full agent runtime.
