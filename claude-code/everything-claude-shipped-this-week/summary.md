# Everything Claude Shipped This Week That Nobody Is Talking About

**Author:** Nav Toor (@heynavtoor)
**Date:** March 20, 2026
**Source:** https://x.com/heynavtoor/status/2035085093191262378

## Key Takeaways

- Anthropic shipped 7 major features in one week (March 13-20, 2026): Dispatch, Channels, Voice mode, 1M context GA, double usage promo, Memory for all, and /loop
- Dispatch creates a phone-to-desktop bridge for Cowork -- send tasks from mobile, get finished work on your computer (currently ~50% reliability on complex tasks)
- Channels connects Claude Code to Telegram/Discord via MCP plugin, positioning it as the simpler alternative to OpenClaw's complex setup
- The 1M token context window is now GA at standard pricing with no multiplier -- entire codebases and full working sessions fit without compaction
- Voice mode uses push-to-talk (spacebar) with 20-language transcription optimized for technical terms, activated via /voice command
- The double usage promotion (2x capacity outside 8AM-2PM ET) signals Anthropic's first experiment with time-based dynamic pricing
- /loop creates session-level cron jobs for monitoring (CI/CD, logs, APIs) that run as long as the session is open

## One-Line Summary

A comprehensive breakdown of Anthropic's most significant product week ever, where seven features collectively shift Claude from a chatbot into always-on infrastructure that works across devices, apps, and schedules.
