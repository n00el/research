# How to Use Claude Dispatch to Run Your AI While You Sleep

**Author:** Nav Toor (@heynavtoor)
**Date:** Mar 19, 2026
**Source:** https://x.com/heynavtoor/status/2034679349157536233

## Key Takeaways

- Claude Dispatch (shipped March 17, 2026) lets you text tasks from your phone to your desktop Claude Cowork instance -- the first consumer AI feature that is genuinely asynchronous
- Setup takes under 5 minutes: update Claude Desktop + mobile app, scan a QR code, toggle file access on
- Information retrieval workflows (email triage, file lookups, meeting prep, research summaries) work at 80%+ reliability; multi-step cross-app workflows are still unreliable at ~50%
- Five reliability tips: build context files (about-me.md, brand-voice.md), use skills for repeated workflows, keep tasks focused on read operations, add Dispatch-specific global instructions, test while near your computer first
- Key limitations: no push notifications, single thread only, no scheduled task management, computer must stay awake, no mobile Cowork configuration
- Dispatch vs. OpenClaw: Dispatch requires a QR code scan while OpenClaw requires a dedicated Mac Mini and extensive configuration -- the accessibility gap is a hundred to one

## One-Line Summary

A comprehensive setup and workflow guide for Claude Dispatch, Anthropic's new feature that turns Cowork into an asynchronous AI assistant you can control from your phone while away from your desk.
