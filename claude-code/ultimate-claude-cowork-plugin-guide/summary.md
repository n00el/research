## Key Takeaways

- Plugins bundle skills, connectors, slash commands, and sub-agents into a single installable package -- they are the "job description" while skills are the individual capabilities
- Anthropic shipped 11 official open-source plugins on January 30, 2026 covering Sales, Marketing, Legal, Finance, Customer Support, Product Management, Data Analysis, Enterprise Search, Biology Research, Productivity, and Plugin Create
- Plugin structure follows a simple directory layout: plugin.json manifest, commands/ for slash commands, skills/ for individual skills, and .mcp.json for connector config
- Key slash commands include /marketing:seo-audit, /sales:call-prep, /legal:contract-review, and /data:visualize
- Connectors (Google Drive, Slack, Notion, Salesforce, etc.) are what let plugins pull live data rather than relying on pasted content
- You can customize any official plugin or build your own from scratch using the Plugin Create plugin

## One-Line Summary

A comprehensive guide to Claude Cowork plugins explaining the plugin-vs-skill distinction, listing all 11 official Anthropic plugins with their slash commands, and providing copy-paste prompts to build custom plugins.
