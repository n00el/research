# How to set up Claude the right way (so you actually stop going back to ChatGPT)

**Author:** Ruben Hassid (@rubenhassid)
**Date:** 2026-02-20
**Source:** https://x.com/rubenhassid/status/2024871775574593798

## Key Takeaways

- Claude in 2026 is not one tool but six: Cowork (desktop app), Model selection (Opus 4.6 + Extended Thinking), Excel add-in, Plugins (11 official skill packs), Artifacts (interactive outputs), and Projects (persistent context folders)
- The biggest mindset shift from ChatGPT: stop writing elaborate prompts and start creating .md/.txt context files that Claude reads from your local folders via Cowork
- Always use Opus 4.6 with Extended Thinking enabled -- it forces Claude to reason before answering, dramatically improving output quality
- Cowork's killer feature is the AskUserQuestion pattern: tell Claude to ask you clarifying questions before executing, turning it into a collaborative partner rather than a one-shot generator
- Claude Plugins disrupted entire industries overnight -- legal software companies lost $285B in market cap in 2 days after launch
- Claude's weaknesses: no image generation (use Gemini), weaker real-time search (use Grok), and Projects has bugs (prefer Cowork + local markdown files)
- Connectors (free on all plans) let Claude read Slack, Google Drive, Notion, Figma and 50+ tools without copy-pasting

## One-Line Summary

A comprehensive beginner-to-intermediate guide positioning Claude's six product surfaces (Cowork, model selection, Excel, Plugins, Artifacts, Projects) as a unified knowledge-work platform that replaces ChatGPT through file-based context rather than prompt engineering.
