A practical setup guide for Anthropic's Claude Cowork feature, aimed at knowledge workers who currently use ChatGPT and want to upgrade their AI-assisted workflow.

## Market Impact

The release of Cowork caused software stocks to lose **$830 billion** in market value over 6 days.

## Claude's 7-Tier Product Line

Chat, Projects, Code, Cowork, Skills, Connectors, and Plugins -- each serving a different use case. Cowork is positioned as the tool for knowledge workers (non-developers).

## Core Philosophy — Abandon Traditional Prompting

Instead of crafting elaborate prompts, users create a structured folder system with context files that Claude reads automatically. This makes prompts short (2-10 lines) while producing highly personalized output.

## Folder Structure (4 Subfolders)

- **ABOUT ME** -- identity, priorities, writing style guide
- **PROJECTS** -- active work organized by project
- **TEMPLATES** -- reusable structural patterns
- **CLAUDE OUTPUTS** -- the only folder where Claude writes files

## AskUserQuestion Feature — The Key Differentiator

Instead of users prompting Claude, Claude prompts the user with interactive forms (clickable options, multi-select, draggable rankings). Users answer in under a minute, then Claude executes with full context.

## Master Prompt Template (80% of Tasks)

> "I want to [TASK] to [SUCCESS CRITERIA]. First, explore CLAUDE COWORK folder. Then ask me questions using AskUserQuestion. I want to refine approach before execution."

## Plugins and Connectors

11 official plugins (Sales, Marketing, Legal, Finance, etc.) plus 50+ connectors (Slack, Google Drive, Notion, Figma). All free.

## Recommended Settings

Always use Opus 4.6 model with Extended thinking enabled. Requires Pro ($20/mo) or Max ($100/mo) account and the desktop app.

## Practical Use Cases

- Newsletter writing with voice-matching
- Consulting deliverables and client reports
- Competitive research and analysis tables
- Automated weekly briefings (scheduled Monday 7am runs)

## Honest Limitations

- Token consumption is heavy; users notice costs within the first week of daily use
- Still in research preview/beta; approximately 10% of complex tasks produce errors
- Requires the desktop app to stay open (no web or mobile version)
- Not suited for quick trivia questions -- only multi-step work

## Comparison with ChatGPT

| Feature | Claude Cowork | ChatGPT |
|---|---|---|
| Context method | File-based, automatic | Paste-and-request |
| Interaction model | Claude asks you questions | You craft prompts |
| File creation | Writes actual files to your system | Text responses only |
| Automation | Scheduled tasks run independently | Manual prompting required |
| Reasoning | Extended thinking (Opus 4.6) | Standard generation |
