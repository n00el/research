# GPT-5.4 Thinking and Pro Rolling Out Now

**Author:** OpenAI (@OpenAI)
**Date:** March 5, 2026
**Source:** https://x.com/OpenAI/status/2029620619743219811

---

GPT-5.4 Thinking and GPT-5.4 Pro are rolling out now in ChatGPT. GPT-5.4 is also now available in the API and Codex. GPT-5.4 brings our advances in reasoning, coding, and agentic workflows into one frontier model.

## Key Details

1. **Two variants announced:** GPT-5.4 Thinking and GPT-5.4 Pro, both rolling out in ChatGPT.
2. **API and Codex availability:** GPT-5.4 is simultaneously available in the OpenAI API and in Codex (OpenAI's agentic coding product).
3. **Unified frontier model:** GPT-5.4 consolidates advances in three areas -- reasoning, coding, and agentic workflows -- into a single model, rather than having separate specialized models for each.
4. **Convergence strategy:** Rather than maintaining separate model lines for reasoning (like the o-series) and general capabilities (like the GPT series), OpenAI is merging these capabilities into one "frontier model."
5. **Agentic workflows as first-class:** The mention of "agentic workflows" as a core capability signals that agent-oriented use cases (multi-step tool use, autonomous task completion) are now a core design goal, not an afterthought.
