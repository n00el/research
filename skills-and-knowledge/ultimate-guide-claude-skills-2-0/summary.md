# Ultimate Guide to Claude Skills 2.0 (the biggest skills update yet)

**Author:** Ole Lehmann (@itsolelehmann)
**Date:** March 10, 2026
**Source:** https://x.com/itsolelehmann/status/2031461162768867622

## Key Takeaways

- Claude Skills 2.0 introduces eval testing: run skills against multiple prompts, get a pass/fail score, and iterate until performance is solid.
- A/B comparison testing lets you benchmark a skill against raw Claude to determine if the skill still adds value or has become counterproductive after model updates.
- The Skill Creator now automatically rewrites skill descriptions to improve triggering accuracy -- Anthropic saw improvement on 5 out of 6 of their own skills.
- Old skills can silently degrade when models improve -- the new A/B testing catches this before it costs you output quality.
- Skills 2.0 is available on Claude.ai, Cowork, and Claude Code via the Skill Creator plugin.

## One-Line Summary

Anthropic's Claude Skills 2.0 adds eval testing, A/B comparison against raw Claude, and automatic description rewriting to fix the three biggest pain points with custom skills: unmeasurable performance, silent degradation after model updates, and unreliable triggering.
