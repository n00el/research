# The Complete Guide to Claude Skills (for non-coders)

**Author:** Khe Hy (@khemaridh)
**Date:** Mar 21, 2026
**Source:** https://x.com/khemaridh/status/2035431416436330624

## Key Takeaways

- Skills are SKILL.md files that codify repeatable workflows -- they are more than reusable prompts because they can bundle examples, scripts, and be shared across teams
- The "skill-creator" skill in Claude Code acts as a meta-tool: you brain-dump your workflow and Claude iteratively interviews you to generate a polished SKILL.md file
- Skills differ from Projects: Projects provide context (files/data), while Skills provide the workflow instructions that operate on that context
- Skills use progressive disclosure for token efficiency -- Claude peeks at the skill index first and only loads full content when relevant
- Skills are composable (combine multiple in one prompt), portable (plain markdown works across LLMs), and becoming an emerging standard beyond Anthropic
- Public skill repositories (Skills.sh, Anthropic GitHub, ClawHub) enable downloading and adapting community-created skills
- Anthropic's legal skill release caused 10%+ stock drops for RELX and Wolters Kluwer, demonstrating skills' disruptive market potential

## One-Line Summary

A non-technical walkthrough showing how Claude Code's skill-creator meta-skill lets anyone turn a brain dump of a repeatable workflow into a reusable, composable, portable SKILL.md file -- the new unit of AI productivity that is disrupting SaaS and professional services.
