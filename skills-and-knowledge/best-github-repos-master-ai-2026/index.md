## The Repos

## 1. Generative AI for Beginners by Microsoft

*[github.com/microsoft/generative-ai-for-beginners](https://github.com/microsoft/generative-ai-for-beginners)*

**21 lessons. Built by Microsoft Cloud Advocates. The best starting point that exists.**

If you are completely new to AI, start here and nowhere else. Microsoft built a 21-lesson course that walks you through everything from understanding what a large language model actually is to building your own generative AI applications. Each lesson is either a "Learn" module that explains a concept or a "Build" module where you write real code. The course covers prompt engineering, chat applications, search apps, image generation, and fine-tuning. It requires basic Python or TypeScript knowledge but assumes zero AI background. What makes this repo special is that it is not a collection of blog posts. It is a structured curriculum designed by professional educators at Microsoft, with quizzes, challenges, and a progression that builds on each lesson. Over 70,000 stars on GitHub. Available in multiple languages. If you have one weekend to start learning AI, spend it here.

## 2. Build a Large Language Model (From Scratch) by Sebastian Raschka

*[github.com/rasbt/LLMs-from-scratch](https://github.com/rasbt/LLMs-from-scratch)*

**Code every component of a ChatGPT-like model. The book that connects all the dots.**

Most people use AI without understanding how it works. This repository is the antidote. Sebastian Raschka, a PhD researcher and bestselling AI author, walks you through implementing a GPT-style language model in PyTorch from the ground up. Not using libraries. Not calling APIs. Actually coding the attention mechanism, the transformer architecture, the training loop, and the fine-tuning pipeline yourself. The repo accompanies his Manning book "Build a Large Language Model (From Scratch)" and includes all the code, diagrams, and exercises. It covers pretraining on a general dataset, fine-tuning for text classification, and creating a model that follows instructions. Everything runs on a regular laptop. Readers consistently say this is the resource that finally made LLMs click for them. If you want to go from "I use ChatGPT" to "I understand how ChatGPT works," this is the path.

## 3. LLM Zoomcamp by DataTalks.Club

*[github.com/DataTalksClub/llm-zoomcamp](https://github.com/DataTalksClub/llm-zoomcamp)*

**A free course focused on real-life applications. Learn by building things that work.**

This is not a theory course. DataTalks.Club designed LLM Zoomcamp around one principle: build real applications from day one. The course teaches you how to use LLMs in practical projects, covering RAG (Retrieval-Augmented Generation), vector databases, orchestration frameworks, and deployment. You build a complete Q&A system, implement search with embeddings, and learn to evaluate LLM outputs in production. What sets this apart is the community. DataTalks.Club runs cohort-based learning with deadlines, peer review, and project submissions. It is the structure of a bootcamp with the cost of zero dollars. If Microsoft's course teaches you the concepts and Raschka's book teaches you the internals, this course teaches you how to ship AI products that people actually use.

## 4. Hands-On Large Language Models

*[github.com/HandsOnLLM/Hands-On-Large-Language-Models](https://github.com/HandsOnLLM/Hands-On-Large-Language-Models)*

**The visual, intuitive guide to understanding LLMs. Perfect for visual learners.**

Some people learn by reading. Others learn by seeing. This repository accompanies the book "Hands-On Large Language Models" and is packed with diagrams, visual explanations, and interactive notebooks that make complex LLM concepts intuitive. It covers the full landscape: how language models represent text, how transformers process information, how attention works, how models are trained, and how to fine-tune and deploy them for real tasks. The notebooks let you experiment with each concept in real time. Change a parameter and see what happens. This is the resource that makes abstract AI concepts concrete. If you have bounced off more technical resources because they felt too dry or too dense, start with this one. The visual approach makes it accessible without dumbing anything down.

## 5. Awesome LLM Apps

*[github.com/Shubhamsaboo/awesome-llm-apps](https://github.com/Shubhamsaboo/awesome-llm-apps)*

**A curated collection of real AI agent applications you can study and build.**

Theory is necessary. But at some point you need to see what finished AI applications actually look like. This repository is a curated collection of real, working AI agent apps built with large language models. Each project is a complete application you can study, run, modify, and learn from. The collection covers AI assistants, research agents, coding tools, data analysis pipelines, and multi-agent systems. What makes this valuable is that every project is a real implementation, not a tutorial exercise. You can see how experienced developers structure AI applications, handle edge cases, manage context, and connect models to external tools. If you have completed a course and are wondering "what do I build now," this repository answers that question with dozens of real examples.

## 6. Prompt Engineering Guide by DAIR.AI

*[github.com/dair-ai/Prompt-Engineering-Guide](https://github.com/dair-ai/Prompt-Engineering-Guide)*

**The most comprehensive prompting resource on the internet. Cited by Google and OpenAI.**

This is the definitive open-source guide to prompt engineering. DAIR.AI created a resource so thorough that Google and OpenAI have both cited it. It covers every major technique: zero-shot, few-shot, chain-of-thought, tree-of-thought, self-consistency, RAG, ReAct, and dozens more. Each technique includes an explanation, research citations, and practical examples. The guide is continuously updated as new techniques emerge, making it one of the few AI learning resources that stays current. It also includes sections on model-specific prompting (what works differently on GPT vs Claude vs Gemini), adversarial prompting and safety, and real-world applications across industries. Whether you are a beginner learning your first prompting technique or an advanced user looking for the latest research, this repo has depth for every level.

## 7. HuggingFace Transformers

*[github.com/huggingface/transformers](https://github.com/huggingface/transformers)*

**The library that powers modern AI. If you want to work with models, you need to know this.**

Hugging Face Transformers is not a course. It is the infrastructure that most AI applications are built on. This library provides access to thousands of pre-trained models for text, image, audio, and multimodal tasks. It supports PyTorch, TensorFlow, and JAX. If you want to load a model, fine-tune it on your data, run inference, or deploy it to production, this is the tool. The reason it belongs on this list is practical: understanding Transformers is not optional if you want to work with AI professionally. The repository includes extensive documentation, tutorials, and example scripts that teach you how to use the library for real tasks. It is the bridge between understanding AI concepts and implementing them. Over 140,000 stars on GitHub. Used by researchers, startups, and enterprises worldwide. Learn this library and you can work with almost any model in existence.

## 8. ML and AI Roadmap 2026 by Daniel Bourke

*[github.com/mrdbourke/machine-learning-roadmap](https://github.com/mrdbourke/machine-learning-roadmap)*

**The map that shows you what to learn and in what order. Prevents tutorial hell.**

The biggest problem beginners face is not a lack of resources. It is too many resources with no clear order. Daniel Bourke created this roadmap to solve that exact problem. It maps the entire machine learning and AI landscape into a visual learning path: what to learn first, what to learn next, what to skip until later, and what resources to use at each stage. The roadmap covers mathematics foundations, programming prerequisites, machine learning fundamentals, deep learning, and applied AI. It is updated for 2026 and reflects the current state of the field, not what was important two years ago. If you are feeling overwhelmed by the number of courses, books, and tutorials available, this roadmap gives you clarity. Print it out. Follow it in order. It will save you months of wandering.

## 9. RAG Techniques by Nir Diamant

*[github.com/NirDiamant/RAG_Techniques](https://github.com/NirDiamant/RAG_Techniques)*

**The deep dive into Retrieval-Augmented Generation. The technique behind every serious AI app.**

RAG is the technique that makes AI applications useful in the real world. Instead of relying solely on what a model was trained on, RAG lets you connect AI to your own data: documents, databases, knowledge bases. This repository is the most comprehensive collection of RAG techniques available. It covers basic retrieval, advanced chunking strategies, hybrid search, re-ranking, multi-step retrieval, and evaluation methods. Each technique includes code, explanations, and benchmarks. If you are building any AI application that needs to work with specific data (a customer support bot, a research assistant, an internal knowledge tool), RAG is the core technique you need to master. This repo takes you from understanding what RAG is to implementing production-grade retrieval systems.

## 10. AI Agents for Beginners by Microsoft

*[github.com/microsoft/ai-agents-for-beginners](https://github.com/microsoft/ai-agents-for-beginners)*

**12 lessons on building AI agents. The skill that companies are hiring for right now.**

AI agents are the biggest shift in AI since ChatGPT launched. Instead of AI that answers questions, agents are AI that takes actions: browsing the web, managing files, coordinating tasks, calling APIs, and completing multi-step workflows autonomously.

Microsoft built this 12-lesson course to teach you the fundamentals from scratch. It covers agent design patterns, tool use, memory systems, handoff mechanisms, failure recovery, and multi-agent collaboration. This is the skill that employers cannot find enough of in 2026. ManpowerGroup's global survey found that AI skills are now the number one hardest skill to find worldwide. Understanding how to build and orchestrate AI agents puts you at the front of that demand curve. If you complete only two repos on this entire list, make them the first one (for foundations) and this one (for the future).

## How to Use This List

Do not try to do all ten at once. That is the fastest path to burnout and the slowest path to competence.

**If you are a complete beginner with no AI background:** Start with #1 (Microsoft Generative AI for Beginners). Then move to #6 (Prompt Engineering Guide). Then #3 (LLM Zoomcamp) to build something real. That sequence takes you from zero to building AI applications in about six weeks.

**If you are a developer who wants to understand how AI works under the hood:** Start with #2 (LLMs from Scratch). Then #7 (HuggingFace Transformers). Then #9 (RAG Techniques). That sequence takes you from user to builder.

**If you want the most career-relevant path:** Start with #1 (Foundations), then #10 (AI Agents), then #9 (RAG). Agents and RAG are the two skills with the highest demand and lowest supply in the job market right now.

**If you are overwhelmed and do not know where to start:** Open #8 (the Roadmap). It will tell you exactly what to learn and in what order. Follow it.

## The Education Is Free. The Advantage Is Not.

Everything on this list is free. Every lesson, every line of code, every tutorial, every exercise. The companies and researchers who built these resources gave them away because open knowledge accelerates the entire field.

But access and action are different things. These repos have millions of stars combined. A tiny fraction of those people completed even the first module. The education is free. The advantage goes to the people who actually do the work.

Pick one repo. Start today. Finish the first lesson before you close this article. That single action puts you ahead of 95% of people who will bookmark this list and never open it again.
