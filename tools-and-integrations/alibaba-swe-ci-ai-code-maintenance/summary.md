# SWE-CI: Alibaba's AI Code Maintenance Benchmark

**Author:** Alex Prompter (@alex_prompter)
**Date:** March 7, 2026
**Source:** https://x.com/alex_prompter/status/2030331477918126286

## Key Takeaways

- SWE-CI tests AI agents on long-term code maintenance (100 codebases, 233 days) — not snapshot tasks
- 75% of AI models break previously working code during maintenance
- Only Claude Opus 4 maintains above 50% zero-regression rate
- Standard benchmarks (HumanEval, SWE-bench) are insufficient for measuring real-world code maintenance ability
- Writing code and maintaining code are fundamentally different skills for AI

## One-Line Summary

Alibaba's SWE-CI benchmark reveals that 75% of AI models break working code during maintenance, with only Claude Opus 4 exceeding a 50% zero-regression rate across 233 days of testing.
